ISHAK / about

Простой issue tracker, который живет в вашем S3

ISHAK хранит проекты и задачи в обычных Markdown-файлах. Никаких скрытых баз данных: только понятная структура, стандартный S3 API и полный контроль над данными.

Как это устроено

Проекты

Каждый проект — это папка в S3: projects/<id>.

Задачи

Каждая задача — Markdown файл: issues/<issueId>/README.md.

Комментарии

Комментарии — отдельные файлы comment_<timestamp>_<role>.md.

Архив

Архивные задачи перемещаются в archive/projects/<projectId>/<issueId>.

Почему это удобно для AI-агентов

Агенту не нужно разбираться в кастомных API. Обычные операции S3 позволяют читать, создавать и обновлять задачи. Markdown дает структуру, понятную и модели, и человеку: заголовки, списки, статусы. Это снижает ошибку интерпретации и ускоряет автоматизацию.

Ваши данные — только у вас

Прозрачность

Все лежит в вашем S3 бакете и доступно привычными инструментами.

Надежность
